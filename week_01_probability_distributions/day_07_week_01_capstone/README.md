# Day 7: Week 1 Capstone

## Project

**The Distribution Profile of Hong Kong Markets**

This capstone combines all Week 1 concepts into a research-style notebook on the Hang Seng Index (`^HSI`).

## Research Questions

1. Does HSI historically reward investors on average?
2. Is HSI a calm or volatile equity market?
3. Can a normal distribution safely describe HSI risk?
4. Are HSI extreme moves more dangerous on the downside?
5. Does HSI diversify a global macro portfolio?

## Main Artifact

- `The_Distribution_Profile_of_Hong_Kong_Markets.ipynb`

The notebook uses real Yahoo Finance data and produces clean tables, charts, and polished written conclusions.

## Key Lessons

- HSI has delivered positive but weak realized compensation relative to volatility.
- Normal distribution assumptions are useful as a baseline, but insufficient for tail-risk analysis.
- HSI has meaningful downside asymmetry and fat-tail risk.
- Average correlations can hide unstable correlation regimes.
- Position sizing should account for volatility, fat tails, skewness, and changing correlations.
