# Day 4: Fat Tails & Why Normal Distribution Fails Markets

## Concept

Fat tails mean extreme events happen more often than a normal distribution predicts.

## Market Connection

Large market crashes are not as rare as naive bell-curve models suggest. This matters for risk management, leverage, and portfolio survival.

## Mini Project

Build a Fat Tail Detector comparing actual extreme days against normal-model expectations.

Run:

```bash
python week_01_probability_distributions/day_04_fat_tails/day_04_fat_tail_detector.py
```

The script prints a clean table and saves QQ-plots under `outputs/`.
